Python:

Python is one of the simplest, but also, one of the most common programming language to exist. It has more than 6 million professional developers, mainly being used in AI development. Not many games are written in Python due to it being such a high-level language, making it quite slow for game loops.

The benchmark of programming has always been making your specific programming language display something to its respective console. In Python, to print something to the console, we use print(). print() is the name of the function, with the text to be displayed, placed within the parentheses. Unless the text is stored within a variable (variables chapter coming soon!) you must enclose the text in double quotes ("text goes here"). For example:

Python
                
print("Hello World!")
Output
Hello World!

To run this code for yourself, you will need Python installed. To check if Python is installed, click Win + R and type in "cmd", then click enter. A command prompt window should open. To check if it's installed, type "py --version". It should now display Python 3.x.x.

Bash
                
py --version

If it comes up with Python 3.0.0 or higher, than you have a suitable Python version installed. You can now skip to the next section.
